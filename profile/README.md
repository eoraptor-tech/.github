## ü¶ñ Who are we?

`rust-eoraptor` is a collective of code explorers with a passion for **Rust**. Our name reflects our philosophy:

* **Primal and Foundational**: We focus on the solid principles that make Rust so powerful.
* **Agile and Fast**: We leverage modern tools, like **AI-assisted coding** (`vaicoding`), to prototype, learn, and build at high speed.
* **Adaptable**: We create versatile software that is built to evolve.

This space is a laboratory for experimenting, sharing knowledge, and developing open-source projects that are both useful and efficient.

## üõ†Ô∏è Our Tech Stack

We primarily focus on the Rust ecosystem, but we also explore technologies that complement it well.

* **Main Language**: Rust

* **Containers**: Docker
